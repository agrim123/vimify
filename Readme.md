# Vim Tricks

## Shortcuts

| Action                            | Combination |
|---------------------------------- |:------------:|
|Toggle tree                        | Ctrl + n |
|Next tab                           | gt |
|Previous tab                       | gT |
| Go to beginning of line           | 0 |
| Go to end                         | $ |
| Move forward one word             | w |
| Move backward one word            | b |
| Move to end of file               | G |
| Move to top of file               |gg |
| Begin a new line above the cursor | O |
| Begin a new line below the cursor | o |
